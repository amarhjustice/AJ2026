# Valentine's Day Website

## Overview
This project is a playful Valentine's Day website designed to engage users with a charming and romantic theme. The main page features a heartfelt question, "Will you be my Valentine?" along with interactive buttons that allow users to respond. The second page expresses gratitude with a sweet message, creating a delightful experience for visitors.

## Setup Instructions
1. Clone the repository to your local machine.
2. Navigate to the project directory.
3. Install dependencies using npm:
   ```
   npm install
   ```
4. Open `index.html` in your web browser to view the website.
